\# 案件管理ミニアプリ



\## 概要



Spring Boot と React / TypeScript を使用した案件管理用のミニアプリです。



\## 作成目的



Java / Spring を使用した既存Webシステムの保守・改修経験をベースに、  

Spring Boot、React、TypeScript を使ったAPI分離型の構成を学習するために作成しました。



\## 使用技術



\- Java 17

\- Spring Boot

\- Spring Web

\- Spring Data JPA

\- H2 Database

\- React

\- TypeScript

\- Vite



\## 主な機能



\- 案件一覧表示

\- 案件登録

\- 案件編集

\- 案件削除

\- ステータス検索

\- 案件詳細表示
